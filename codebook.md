## Values
### subject values:
THere are 30 sabjects encoded with ids: 1..30

### activity values:
Activity names are taken from file: activity_labels.txt
* WALKING
* WALKING_UPSTAIRS
* WALKING_DOWNSTAIRS
* SITTING
* STANDING
* LAYING

### Feature variable renges:
Accoding to original data description features are normalised and bounded to [-1,1]
Avereges are calculated on provided values without any other modififcations.
